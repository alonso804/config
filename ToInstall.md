## NodeJS 
[Github](https://github.com/nvm-sh/nvm#installing-and-updating)
```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/<nvm_version>/install.sh | bash
nvm ls-remote 
nvm install <version>
```

Variables in fish
```fish
set PATH /home/user/.nvm/versions/node/<version>/bin $PATH
```
